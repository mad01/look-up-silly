# App Icon Placeholders

Add your app icons here with the following filenames:

- icon-20@2x.png (40x40)
- icon-20@3x.png (60x60)
- icon-29@2x.png (58x58)
- icon-29@3x.png (87x87)
- icon-40@2x.png (80x80)
- icon-40@3x.png (120x120)
- icon-60@2x.png (120x120)
- icon-60@3x.png (180x180)
- icon-1024.png (1024x1024)

For now, you can use SF Symbols or generate icons using:
- https://www.appicon.co/
- Figma
- Sketch
- Or any image editor

The app icon should represent "looking up" from your phone - maybe an arrow pointing up or an eye looking up.
